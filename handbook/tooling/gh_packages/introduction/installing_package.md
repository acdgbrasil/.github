# Instalando um pacote

Você pode instalar um pacote do GitHub Packages e usá-lo como uma dependência no seu próprio projeto.

<!-- 2148AF7B-5FF8-4B28-A808-D692FEE2225A -->

## Sobre a instalação do pacote

Você pode pesquisar em GitHub para encontrar pacotes em GitHub Packages que você pode instalar no seu próprio projeto. Para saber mais, confira [Pesquisar pacotes](/pt/search-github/searching-on-github/searching-for-packages).

Depois de encontrar um pacote, você pode ler a descrição e as instruções de instalação e utilização na página de pacotes.

## Instalando um pacote

Você pode instalar um pacote do GitHub Packages usando qualquer cliente do pacote compatível, seguindo as mesmas diretrizes gerais.

1. Efetue a autenticação com GitHub Packages usando as instruções para seu cliente de pacote. Para saber mais, confira [Introdução ao GitHub Packages](/pt/packages/learn-github-packages/introduction-to-github-packages#authenticating-to-github-packages).
2. Instale o pacote usando as instruções do cliente do pacote.

Para obter instruções específicas do cliente do pacote, confira [Trabalhar com um registro do GitHub Packages](/pt/packages/working-with-a-github-packages-registry).
