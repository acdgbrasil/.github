# Trabalhando com o registro do Docker

O registro do Docker foi substituído pelo Container registry.

<!-- Main versioning block. Short page for dotcom -->

O registro do Docker do GitHub (que usava o namespace `docker.pkg.github.com`) foi substituído pelo Container registry (que usa o namespace `https://ghcr.io`). O Container registry oferece benefícios como, por exemplo, permissões granulares e otimização de armazenamento para imagens Docker.

As imagens do Docker armazenadas anteriormente no registro do Docker estão sendo automaticamente transferidas para Container registry. Para saber mais, confira [Fazendo a migração para o Registro do Contêiner a partir do Registro Docker](/pt/packages/working-with-a-github-packages-registry/migrating-to-the-container-registry-from-the-docker-registry) e [Trabalhando com o registro do Contêiner](/pt/packages/working-with-a-github-packages-registry/working-with-the-container-registry).
